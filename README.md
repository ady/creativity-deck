# Creativity Deck

A stateless, single-file web app for drawing random prompts from a 52-card creativity deck.

The deck has 4 suits, 13 cards each:

- Methods & Mechanisms
- Seeing Differently
- Play & Experimentation
- Mindset & Momentum

Each card has a title and a longer description. Hit "Draw a Card" to get a random one (card ranks 1, 11, 12, 13 are shown as A, J, Q, K).

## Usage

Open [`index.html`](index.html) directly in any browser — no server, build step, or dependencies required. All card data is embedded in the page.
